# LHI Lecce NFT - Mint Frontend

Questo è il frontend pubblico del progetto **LHI Lecce Medical Hotel Italia**, sviluppato per il minting di NFT dedicati alla ricerca medica su **Fibromialgia e Depressione**.

## 🌍 Descrizione del progetto

Il progetto è basato su **blockchain Polygon (MATIC-Ethereum)** e utilizza uno smart contract per consentire l'acquisto di NFT da parte di utenti connessi tramite Metamask.

Ogni NFT rappresenta un contributo al programma di **Crownfounding International**, sostenuto da **LHI Lecce Medical Hub**, attivo nella ricerca scientifica e terapeutica.

- 💻 **Pagina online del frontend:** https://avvstancamarcello.github.io/lhi-nft-frontend/

## 🚀 Funzionalità

- Selezione dell'NFT tramite 24 pulsanti numerici (ID: 1, 2, 3, ..., 100)
- Visualizzazione dinamica delle anteprime da `images/`
- Integrazione con **Metamask** (Ethereum provider)
- Invio di transazioni al contratto smart per effettuare il mint

## 📁 Struttura

- `index.html`: interfaccia utente del frontend
- `images/`: contiene le 24 immagini `.jpg` delle anteprime NFT
- `README.md`: documentazione del repository

## 🛠 Requisiti

- Estensione **Metamask** installata nel browser
- Connessione al network **Polygon (Mainnet)**
